HOUSE OF NOLUHLE — LOGO SET
============================

All SVG files have transparent backgrounds and the text is converted
to outlines (Playfair Display), so they render identically on any
device with no font installation needed.

BRAND COLOURS
  Charcoal  #2B2B2B  (primary text/mark)
  Gold      #C9A227  (rings and accents)
  Cream     #FAF6F0  (reverse mark, backgrounds)

FILES & WHERE TO USE THEM
  logo-primary.svg        Main logo for light backgrounds — website header
  logo-reverse.svg        Cream version for dark backgrounds — footer, dark banners
  monogram-primary.svg    Square N mark — social profile pictures, watermarks
  monogram-reverse.svg    Cream monogram for dark backgrounds
  favicon.svg             Simplified icon for browser tabs
  favicon-32.png          Classic 32px favicon fallback
  apple-touch-icon-180.png  iPhone home-screen icon
  logo-primary-1600w.png  High-res transparent PNG — invoices, docs, printing
  monogram-1024.png       High-res monogram PNG — Instagram/TikTok profile

WEBSITE SETUP (add inside <head>)
  <link rel="icon" href="/favicon.svg" type="image/svg+xml">
  <link rel="icon" href="/favicon-32.png" sizes="32x32">
  <link rel="apple-touch-icon" href="/apple-touch-icon-180.png">

TIP: In your site header, use the SVG at 180–260px wide.
Since SVGs are vector, they stay crisp at any size.
